A full-stack fitness tracking application for creating workout programs, tracking sessions, and monitoring progress. 


## **Features**

- User authentication (register / login / logout)

* Create and manage workout programs

* Program-based workout sessions

- Manual exercise logging

- Personal Record (PR) tracking

- Exercise history with filtering

- Progress visualization graphs

- Responsive UI

- Unit testing + End-to-End testing

+ CI/CD pipeline with automated deployment

## **Tech Stack**

### **Frontend**

- React

* Vite

+ Tailwind CSS

### **Backend / Database**

- Supabase (Auth + PostgreSQL)

### **Testing**

- Vitest (Unit Tests)

- Robot Framework

- Selenium (E2E Tests)

### **CI/CD & DevOps**

- GitHub Actions

- GitHub Pages Deployment

- Headless Chrome testing in CI

## **This project includes both unit tests and end-to-end tests.**

### Unit Tests

- Component logic validation

- Authentication behavior

- Navigation visibility

- Program and session functionality

### End-to-End Tests

- Login / Logout flows

- Protected route access

- Exercise logging

- Program session flows

### CI runs:

- Smoke tests on Pull Requests

- Full test suite on merges to main
